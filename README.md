## Getting Started


First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Make sure you are running Node.js version 18 or higher.

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
